* YoloV3_self
** 简介
   YoloV3的pytorch复现
   1. Dataset: [[https://aistudio.baidu.com/aistudio/datasetdetail/20495][十分类交通标志物数据集]]
   2。Backbone: darknet53
   3. Loss function:
      - BCE: x、y、confidence、class
      - MSE: w、h
   4. Features：argparse、gpu加速、学习率更新、断点续训、类内NMS...
